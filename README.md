# fileOperationTool
swift封装的文件操作工具类

封装了创建文件、文件夹, 计算文件大小, 移动文件,写入字符串、字典等常用文件操作

Encapsulates the create a file, folder, file size, move files, written string, dictionaries and other commonly used file operations
